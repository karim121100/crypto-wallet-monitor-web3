# Crypto Wallet Monitor (Web3)

This is a simple Web3 project to check crypto wallet balances and recent transactions.

The goal of this project is to help beginners understand how blockchain data is fetched and shown in an app.

## What this project does

- Connects to a public blockchain API  
- Reads wallet balance  
- Reads recent transactions  
- Shows data in the console  
- Works with Ethereum-style addresses  

## Why this project is useful

If you want to learn:
- How Web3 apps talk to the blockchain
- How wallet addresses work
- How APIs return on-chain data
- How to structure a simple crypto project

This is a good starting point.

## Tech used

- JavaScript
- Node.js
- Web3 API (example format)
- Basic blockchain concepts

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `config.js` → API and network settings  
- `wallet.js` → Wallet functions  
- `balance.js` → Balance checker  
- `transactions.js` → Transaction history  
- `utils.js` → Helper functions  
- `logger.js` → Console logger  
- `network.js` → Blockchain network info  
- `sample.env` → Example API config  
- `README.md` → This file

## Notes

This project is for learning only.  
Do not use real private keys in this app.
